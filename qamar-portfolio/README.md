# Qamar Abbas Portfolio

Premium modern portfolio website for Qamar Abbas - Digital Marketing, Generative AI, and eCommerce Specialist.

## Features
- Fully responsive design
- Smooth animations and typing effect
- Mobile hamburger menu
- Contact form
- Modern gradient design
- Scroll-triggered animations

## How to Run
1. Open `index.html` in any modern web browser
2. Or use Live Server extension in VSCode
3. Navigate sections using navbar

No dependencies required - all assets via CDN.
